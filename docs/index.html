MP3Joiner

Jednoduchá macOS aplikace pro spojování MP3 souborů

📥 Stáhnout MP3Joiner v1.0.0 (DMG - doporučeno)

Velikost: 1.5 MB | macOS 14.0+ | Profesionální DMG

Základní DMG | ZIP verze

🎵 Spojování MP3

Spojujte MP3 soubory pomocí FFmpeg s vysokou kvalitou

🖱️ Drag & Drop

Přetahujte soubory přímo z Finderu do aplikace

📋 Seřazování

Seřaďte soubory šipkami nahoru/dolů s výběrem

🎨 Moderní UI

Krásné SwiftUI rozhraní s vlastní ikonou

📋 Požadavky

  • macOS 14.0 nebo novější
  • FFmpeg - nainstalujte pomocí: brew install ffmpeg

🚀 Instalace (DMG - doporučeno)

  1. Stáhněte MP3Joiner-1.0.0-professional.dmg
  2. Dvakrát klikněte na DMG soubor (mount)
  3. Přetáhněte MP3Joiner.app do Applications složky
  4. Eject DMG (odpojte)
  5. Spusťte aplikaci z Applications

📦 Alternativní instalace (ZIP)

  1. Stáhněte MP3Joiner-1.0.0-distribution.zip
  2. Rozbalte ZIP soubor
  3. Pravé tlačítko na MP3Joiner.app → "Otevřít"
  4. Přetáhněte do složky Applications
  5. Spusťte aplikaci

🛡️ Řešení problému s ověřením

Pokud se zobrazí: "Společnosti Apple se nepodařilo ověřit..."

DMG soubory mají méně problémů než ZIP!

  1. Pravé tlačítko na DMG soubor
  2. Vyberte "Otevřít"
  3. Klikněte "Otevřít" v dialogu
