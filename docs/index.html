MP3Joiner

Jednoduchá macOS aplikace pro spojování MP3 souborů

📥 Stáhnout MP3Joiner

Velikost: 1.5 MB | macOS 14.0+ | Apple ověřená aplikace

Základní DMG | ZIP verze

🎵 Spojování MP3

Spojujte MP3 soubory pomocí FFmpeg s vysokou kvalitou

🖱️ Drag & Drop

Přetahujte soubory přímo z Finderu do aplikace

📋 Seřazování

Seřaďte soubory šipkami nahoru/dolů s výběrem

🎨 Moderní UI

Krásné SwiftUI rozhraní s vlastní ikonou

📋 Požadavky

  • macOS 14.0 nebo novější
  • FFmpeg - nainstalujte pomocí: brew install ffmpeg

🚀 Instalace (NOTARIZOVANÁ VERZE - doporučeno)

  1. Stáhněte MP3Joiner-1.0.0-notarized.dmg
  2. Dvakrát klikněte na DMG soubor (mount)
  3. Přetáhněte MP3Joiner.app do Applications složky
  4. Eject DMG (odpojte)
  5. Spusťte aplikaci z Applications

📦 Alternativní instalace

  1. Stáhněte jinou verzi (profesionální DMG, základní DMG, nebo ZIP)
  2. Postupujte podle stejných kroků
  3. Pro ZIP: rozbalte a pravým tlačítkem → "Otevřít"
